# News App

## Running the project

1. Clone the project.
2. Run `npm install`
3. Create a `.env.local` file.

```
{
GITHUB_ID=""
GITHUB_SECRET=""
NEXT_PUBLIC_OPENAI_API_KEY=""
}
```

Now run the server:

```
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
